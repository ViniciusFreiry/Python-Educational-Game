# 🎮 Python Educational Game -- Neon Fraction Scanner

An interactive **educational 3D game** built with Python and OpenGL,
designed to help players understand **fractions and proportional
reasoning** through gameplay.

------------------------------------------------------------------------

## 🚀 Overview

**Neon Fraction Scanner** is a game where the player must align a
scanner with the correct position of a fraction on a number line.

Instead of solving math traditionally, the player **visualizes fractions
spatially**, reinforcing intuition and learning through interaction.

------------------------------------------------------------------------

## 🧠 Learning Objectives

-   Understand fractions as proportions
-   Visualize fractions on a number line
-   Improve estimation and approximation skills
-   Reinforce math concepts through gameplay

------------------------------------------------------------------------

## 🎮 Gameplay

-   Move the scanner using:
    -   `A` → Move left
    -   `D` → Move right
-   Press `SPACE` to validate your position
-   Match the correct fraction position to score points

### 🎯 Scoring System

-   ✅ Accurate hit → +10 points + bonus time\
-   🟢 Perfect alignment → more bonus time\
-   ❌ Miss → lose a life\
-   ⏱️ Time decreases constantly → game over when it reaches 0

------------------------------------------------------------------------

## ✨ Features

-   🧮 Educational gameplay focused on fractions

-   ⚡ Real-time feedback system
-   🎆 Particle effects on success

## 📦 Installation

### 1. Clone the repository

``` bash
git clone https://github.com/ViniciusFreiry/Python-Educational-Game.git
cd Python-Educational-Game
```

### 2. Install dependencies

``` bash
pip install PyOpenGL PyOpenGL_accelerate
```

------------------------------------------------------------------------

## ▶️ Running the Game

``` bash
python EducationalGame.py
```

> Make sure your system supports OpenGL.
